## 題目：Tetris

## 組員：
• B11207040 林信儒

• B11207140 劉胤言
## 玩法：
• 使用左右方向鍵控制左右移動

• 使用上方向鍵控制方塊右轉

• 使用Z鍵控制方塊左轉

• 使用下方向鍵加快方塊下落

• 使用空白鍵讓方塊直接到底

• 使用P鍵暫停遊戲(暫停後僅能重開或離開)

• 使用C鍵儲存方塊及拿取儲存之方塊

• 使用空白鍵讓方塊直接到底

## 功能：
• 顯示起始畫面

• 顯示遊戲畫面

• 顯示當前分數及等級

• 顯示儲存及下一個的方塊

• 消除方塊並整體下降一格
